English
Installation To run this project locally, ensure you have Python installed. Then, install the required dependencies using the following command:

Bash
pip install -r requirements.txt
Note for Linux users: You may also need to install python3-tk via your package manager (e.g., sudo apt install python3-tk).

Spanish (Argentine)

Nota para usuarios de Linux: Es posible que también necesites instalar python3-tk desde el gestor de paquetes (ej: sudo apt install python3-tk).

Tip del desarrollador:
Con Python, lo ideal es no instalar todo de forma global en la computadora, sino usar un Entorno Virtual (venv). Esto evita que las versiones de las librerías choquen entre distintos proyectos.

Si querés probarlo en tu Lubuntu o en Windows, los pasos serían:

Crear el entorno: python -m venv venv

Activarlo:
Windows: .\venv\Scripts\activate
Linux: source venv/bin/activate

Instalar los requisitos: pip install -r requirements.txt
